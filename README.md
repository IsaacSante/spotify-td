# spotify-td
Import TOX file into your TD project.
Hit the open spotify pulse button to get the websocket connection started.
<img src="./markdown_assets/parameters.png" alt="Parameters" width="600">

This only works for song with lyrics. 
You can have the app in the background and control songs from your phone for the best experience. 

On the spotify app end enable the full screen lyrics.
